실제 HTML 파일을 jsdom으로 렌더링하고 버튼 클릭 이벤트를 그대로 구동해 관측 검증을 완료했습니다.

**검증 방식**: `counter.html` 파일 자체를 로드 → 실제 DOM의 `#plus/#minus/#reset` 버튼에 click 이벤트 디스패치 → `#count` 표시값 관측 (JS 재구현이 아닌 파일의 실제 스크립트 실행).

**관측 증거**:
| 동작 | `#count` 관측값 | 기대 |
|------|------|------|
| 초기 | `0` | ✓ |
| +3 클릭 | `3` | ✓ |
| −1 클릭 | `2` | ✓ |
| 초기화 | `0` | ✓ |
| 0에서 −1 | `-1` | ✓ |
| 필수 요소 4종 존재 | OK | ✓ |

모든 인터랙션이 실제 파일에서 기대대로 동작함을 확인했습니다. 브라우저에서 `counter.html`을 열면 동일하게 작동합니다.
